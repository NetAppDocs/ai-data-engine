= Keep your AI Data Engine system up to date
:icons: font
:imagesdir: ../media/

[.lead]
To maintain optimal performance, security, and access to the latest features, there are several components of an AI Data Engine system that you need to keep up to date. You might need to update these components after a new deployment when newer software or firmware is available, when you add or replace nodes, or periodically as new features and improvements are released.

== AIDE system components
DCN firmware and AIDE Console software updates are distributed as a single, separate package (`dcn-image.tgz`) and are not embedded in ONTAP images.

=== ONTAP Software
ONTAP is the operating system that runs on NetApp storage systems, including those used in AI Data Engine deployments. Keeping ONTAP up to date is critical for maintaining system stability, security, and compatibility with AIDE system components that rely on ONTAP but are updated separately.

=== DCN firmware updates
Firmware updates are essential for the proper functioning of hardware components within the AI Data Engine system, including data compute nodes (DCNs). This includes updates for storage controllers, network interfaces, and other critical hardware.

=== AI Data Engine software
The AI Data Engine Console software, including vectorization and guardrail capabilities, requires regular updates that can include new features, performance enhancements, and bug fixes.

== Compatibility matrix
It's important to ensure compatibility between the software and firmware of all AI Data Engine system components, including ONTAP, DCN firmware, and AI Data Engine software.

|===
| ONTAP Version | Compatible AIDE Console and DCN Firmware Versions 
| 9.18.1 and later | AIDE Console 1.0.x, DCN Firmware 1.0.x
|===

== Revert limitations
Revert functionality is not supported for DCN firmware, AI Data Engine Console software updates, or ONTAP software for AFX storage systems. After an update or upgrade is installed, you cannot revert to a previous version. Before making any updates or upgrades, ensure you have reviewed the release notes and compatibility requirements for each component.

== Understand the update process
AI Data Engine DCN and AIDE Console software updates can be managed through ONTAP System Manager, which provides options for partially automatic or manual updates.

Administrators cannot enable fully automatic updates of DCN software. By default, AIDE admins will be notified of an available update, but the update will not be automatically downloaded or installed. Admins must acknowledge the pending update before it is downloaded and installed.
