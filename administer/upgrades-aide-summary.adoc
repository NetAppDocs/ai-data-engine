= Keep your AI Data Engine system up to date
:icons: font
:imagesdir: ../media/

[.lead]
Keep AI Data Engine (AIDE) system components up to date to maintain optimal performance and access to new features. Update components after deployment, when new software or firmware becomes available, when you add or replace nodes, or periodically for feature updates.

== AIDE system components
There are two main components that require updates in an AI Data Engine system: ONTAP software and AI Data Engine software, which includes DCN firmware.

=== ONTAP Software
ONTAP is the operating system that runs on NetApp storage systems, including those used in AI Data Engine deployments. Keep ONTAP current to preserve system stability, security, and compatibility with AIDE components. AIDE components are updated separately.

=== AI Data Engine software updates
AIDE Console software and DCN firmware updates are distributed together as a single package (.tgz) and are not embedded in ONTAP images.

.DCN firmware updates
Update firmware to ensure the proper functioning of hardware components within the AI Data Engine system, including data compute nodes (DCNs), storage controllers, network interfaces, and other critical devices.

.AI Data Engine Console software
Update the AI Data Engine Console software regularly to obtain new features, performance improvements, and bug fixes. AIDE software includes the AI Data Engine Console, vectorization, and guardrail components.

== Understand the update process
AI Data Engine software updates can be managed through ONTAP System Manager.

AIDE does not fully support ONTAP automatic software updates functionality. If you use ONTAP automatic updates, the system can notify admins about available updates but not make the updates for them. Admins must complete the process manually after notification.

Release types and scope:

* ONTAP major releases and AIDE major releases introduce new features, APIs, or changes that affect ONTAP integration.
* ONTAP patch releases and AIDE update releases contain fixes and updates that do not impact ONTAP integration.

== Compatibility matrix
Ensure compatibility between ONTAP, DCN firmware, and AI Data Engine software when planning updates.

AI Data Engine software is released as "U" releases. An AIDE major release is a "U0" release and subsequent minor releases will be "U1" and later.

=== ONTAP and AIDE compatibility
[cols="1,1",options="header"]
|===
| AIDE Releases | Supported ONTAP Versions

| 9.18.1 U0 | 9.18.1 GA and all 9.18.1 Px 
//(and the next three major ONTAP 9.xx.x releases)
| 9.18.1 U1 and later | 9.18.1 GA and all 9.18.1 Px 
//(and the next three major ONTAP 9.xx.x releases)
|===

[NOTE]
====
"Px" denotes all ONTAP patch releases in major version (for example, 9.18.1 P1, 9.18.2, and others).
====

== Revert limitations
AIDE systems do not support revert operations for DCN firmware, AIDE software updates, or ONTAP on AFX storage systems. After you install an update or upgrade, you cannot revert to a previous version. Review <<Compatibility matrix,compatibility requirements>> before you update or upgrade.
