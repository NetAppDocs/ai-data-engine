= Update DCN firmware and AI Data Engine Console software
:icons: font
:imagesdir: ../media/

[.lead]
You can update data compute node (DCN) firmware, AI Data Engine Console software including vectorization and guardrails functionality, and other system files on your AI Data Engine system using ONTAP System Manager.

AIDE does not fully support ONTAP automatic software updates functionality. If you use ONTAP automatic updates, the system can notify admins about available updates but not make the updates for them. Admins must complete the process manually after notification.

.About this task

The combined AI Data Engine software package is significantly larger than a typical ONTAP update package (AIDE package is approximately 40GB). Plan for longer upload and installation times when updating AIDE software.

.Before you begin

* You need _storage administrator_ privileges to update DCN firmware and AI Data Engine Console software.
* You need your NetApp Support Site credentials for your active account.
* link:upgrades-aide-summary.html#compatibility-matrix[Ensure compatibility between ONTAP, DCN firmware, and AI Data Engine software when planning updates].

IMPORTANT: Revert functionality is not supported for DCN firmware or AI Data Engine software updates. After an update is installed, you cannot revert to a previous version.

.Steps

. https://mysupport.netapp.com/site/products/all/details/aide/downloads-tab[Download the combined DCN firmware and AIDE software update file to a local client^].
. In System Manager, select *Cluster > Settings > Software updates*.
. Next to *Software updates*, select image:icon_arrow.gif[blue arrow pointing to the right].
. Under AI Data Engine updates, select *Add AI Data Engine software files* and select the update package.
. After package upload is complete, select *Update* to begin installing updates on the DCN nodes.

.Result
The DCNs are updated with AI Data Engine software and the updated version is displayed for each node.
