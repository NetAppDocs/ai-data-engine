= AIDE components and data scientists
:icons: font
:imagesdir: ../media/

[.lead]
As a data scientist, you use the AI Data Engine Console to search for relevant data, create and refine data collections, perform semantic searches, and integrate retrieval endpoints into AI/ML workflows. Your role focuses on leveraging curated datasets for analysis, model training, and GenAI applications.

== Data scientist component access

[cols="2h,3", options="header"]
|===
|Component
|Data scientist access

|System Manager
|No access

|AI Data Engine Console
|Manage (create, edit, delete)

|REST API
|Manage (create, edit, delete)

|Workspaces
|View/Use (read-only, consume for workflows)

|Data collections
|Manage (create, edit, delete)

|Guardrails
|View/Enforce (read-only, operate within)

|Metadata catalog
|Query/Use (consume for workflows)

|Vector database
|Use Search (query, consume for workflows)

|Classifiers
|Use (consume classified data)

|===

== AIDE system components

=== Workspaces

You receive access to workspaces relevant to research or modeling tasks. You use the AI Data Engine Console to search for and understand available data, focusing on files and objects needed for analysis or model training.

=== Metadata catalog

You use the metadata catalog to search for and understand available data, focusing on files and objects needed for analysis or model training.

=== Classifiers

You benefit from pre-classified data, ensuring that only compliant and relevant data is used in analysis and modeling.

=== Data collections

You select and use data collections for experiments, model training, or inference. You annotate, version, or refine collections as analysis progresses. You use collections as the basis for semantic search and GenAI workflows.

=== Vector database

You use semantic search APIs to find relevant data in collections, leveraging embeddings generated by automated pipelines. You integrate retrieval endpoints into GenAI or RAG workflows for context-aware model responses.

=== Guardrails

You operate within the constraints of guardrails attached to workspaces and data collections. You rely on guardrails for automated classification, redaction, and protection of sensitive data, ensuring compliance without manual intervention.

=== Retrieval endpoint (RAG endpoint)

You use retrieval endpoints to power GenAI and RAG workflows, enabling models to connect to data and provide context-aware responses.
