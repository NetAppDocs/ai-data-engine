= Learn how AI Data Engine data scientists work with AIDE components
:icons: font
:imagesdir: ../media/

[.lead]
As a data scientist, you use the AI Data Engine Console to search for relevant data, create and refine data collections, perform semantic searches, and integrate retrieval endpoints into AI/ML workflows. Your role focuses on leveraging curated datasets for analysis, model training, and GenAI applications.

== Data scientist component access

[cols="2h,3,4", options="header"]
|===
h|Component
h|Data scientist access
h|Your workflow

|AI Data Engine Console
|Manage (create, edit, delete)
|The AI Data Engine Console is your primary interface for data exploration, refining and versioning collections, and connecting curated datasets to analysis and modeling workflows.

|ONTAP REST API
|Manage (create, edit, delete)
|You use the REST API to programmatically access data collections, run vector search queries, and integrate retrieval endpoints into AI/ML applications and agentic frameworks.

|Workspaces
|View/use (read-only)
|You search assigned workspaces to locate files and objects relevant to specific research or modeling tasks.

|Data collections
|Manage (create, edit, delete)
|You select, annotate, version, and refine collections for experiments, model training, or inference, and you use those collections as the basis for semantic search and GenAI workflows.

|Metadata catalog
|Query/use (consume for workflows)
|You search and filter metadata to locate files and objects needed for analysis or model training, relying on the catalog structure that has been built and maintained by data engineers.

|Vector database
|Use/search (query only)
|You run semantic search queries against embeddings generated by data engineer pipelines and integrate retrieval results into GenAI or RAG workflows for context-aware model responses.

|Classifiers
|Use (consume classified data)
|You consume pre-classified data to ensure that only compliant and relevant content is used in your analysis and modeling; you cannot modify classifier configuration.

|===
