= AI Data Engine Installation Requirements
:icons: font
:imagesdir: ../media/

[.lead]
AIDE requires specific hardware, software, networking, and operational prerequisites for successful deployment and operation.

== Hardware requirements

=== Data Compute Nodes (DCNs)

* Minimum: 64 CPU cores, 1TB RAM, GPU acceleration (e.g., NVIDIA L4 or higher)
* Must be certified for NetApp ONTAP AI Data Platform
* Direct connection to cluster/storage switches

=== ONTAP Nodes

* Must meet ONTAP cluster hardware specifications
* Sufficient capacity for PB-scale storage

=== Networking

* Redundant cluster/storage switches (minimum 100GbE)
* Sufficient switch ports for ONTAP and DCN nodes
* Isolated VLAN/IPspace for DCN-ONTAP communication

=== Storage Shelves

* NVMe-oF shelves with SSDs
* Redundant power, cooling, and network interfaces

== Software requirements

=== Operating System (DCN)

* Linux (CLP/Debian), chosen for AI/ML package compatibility
* Container orchestration: Kubernetes (RKE or equivalent)
* Document database: MongoDB (for metadata catalog)
* Vector database: Milvus (for embeddings)
* AI/ML frameworks: PyTorch, TensorFlow, and other relevant libraries

=== ONTAP Software

* ONTAP version 9.18.1 or higher (required for AIDE features)
* Separate DCN software image (`ontap_aidp.tgz`)
* Licensing: Ontap One (includes Metadata Engine Basic); AIPVS Vectorization and Guardrails license required for each DCN GPU

== Networking requirements

=== Cluster Network

* Dedicated VLAN/IPspace for DCN-ONTAP communication
* Sufficient IP addresses for all nodes plus load-balanced interfaces

=== Intercluster Connectivity

* Intercluster interfaces for peering with remote ONTAP clusters
* SnapMirror for data replication

== Other requirements

=== High Availability

* Redundant hardware and network paths
* Support for rolling upgrades and failover

=== Security

* TLS/SSL for all management and data traffic
* OIDC/OAuth 2.0 for user authentication
* RBAC for user and resource access

=== Monitoring and Logging

* Integration with ONTAP EMS and AutoSupport
* Prometheus/Grafana for metrics and dashboards

=== Installation and Upgrade

* System Manager for cluster setup and management
* Manual upload of DCN software images (auto-update may be available in future releases)
