= Learn about AI Data Engine
:icons: font
:imagesdir: ../media/

[.lead]
The NetApp AI Data Engine (AIDE) is an enterprise-grade platform designed to accelerate and simplify AI-driven data processing, management, and governance. AIDE can help transform large amounts of unstructured data into structured, AI-ready datasets. It is engineered to meet the demands of modern machine learning (ML) and generative AI (GenAI) workloads, supporting both traditional IT operations and new AI-centric roles.

== AIDE addresses AI challenges

AIDE addresses several key challenges faced by organizations deploying AI at scale:

* Data Discovery and Governance: Centralizes metadata management, making it easy to find, classify, and govern datasets across the enterprise.
* Scalable AI Data Processing: Automates and optimizes data pipelines for AI/ML workloads, including retrieval-augmented generation (RAG) and embedding creation.
* Secure, Multi-Tenant Operations: Ensures robust data isolation, access control, and compliance for multiple teams or projects.
* Operational Simplicity: Integrates with familiar NetApp management tools, reducing complexity for administrators and enabling self-service for data engineers and scientists.

== High-Level Design Characteristics

AI Data Engine is built on several key design principles:

* Disaggregated architecture: Separates compute (Data Compute Nodes, or DCNs) from storage (ONTAP Nodes), allowing independent scaling and optimization.
* Microservices-based services: Uses Kubernetes to orchestrate modular, resilient services for metadata cataloging, vector search, and infrastructure management.
* Enterprise-grade security: Implements encryption, role-based access control (RBAC), and auditing across all data and metadata.
* Multi-protocol data access: Supports NFS, SMB, and S3 for flexible data ingestion and retrieval.
* Automated data pipelines: Tracks data changes, creates embeddings, and manages vector databases for AI applications.
