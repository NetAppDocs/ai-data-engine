= Learn how AI Data Engine data engineers work with AIDE components
:icons: font
:imagesdir: ../media/

[.lead]
As a data engineer, you use the AI Data Engine Console to explore workspaces, create data collections, manage embedding pipelines, and integrate AI/ML workflows. Your role focuses on transforming raw data into AI-ready datasets.

== Data engineer component access

[cols="2h,3,4", options="header"]
|===
h|Component
h|Data engineer access
h|Your workflow

|AI Data Engine Console
|Manage (create, edit, delete)
|The AI Data Engine Console is your primary interface for all day-to-day tasks, including data discovery, collection management, pipeline configuration, and publishing RAG endpoints.

|ONTAP REST API
|Manage (create, edit, delete)
|You use the REST API to automate collection lifecycle operations, trigger embedding pipelines, and integrate data workflows with external tools programmatically.

|Workspaces
|View/use (read-only)
|You explore assigned workspaces to identify and understand available data sources before building collections.

|Data collections
|Manage (create, edit, delete)
|You build data collections by selecting and filtering source data using tags, classification, and other attributes, and you manage the full collection lifecycle from creation through publishing as RAG endpoints for AI/ML use.

|Metadata catalog
|Query/use (consume for workflows)
|You use the metadata catalog to evaluate and select data sources for ingestion, running queries to locate relevant files and confirm they meet the requirements of the collections you are building.

|Vector database
|Manage embeddings/search (create, configure, query)
|You trigger embedding pipelines, monitor vectorization status, configure chunking and embedding parameters, and integrate retrieval endpoints into AI applications via the API.

|Classifiers
|Use (consume classified data)
|You use classification results to annotate and tag source data during collection preparation, ensuring that content entering your pipelines is properly labeled for downstream AI workflows.

|===
