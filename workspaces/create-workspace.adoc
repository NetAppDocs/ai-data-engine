= Set up a workspace and add a data collection in AI Data Engine
:icons: font
:imagesdir: ../media/

[.lead]
As a storage administrator, you use System Manager to create workspaces, view workspace details, and add data collections to support AI workloads. Workspaces are logical containers that help organize data containers and collections, making it easier to manage and prepare data for AI applications.

.Before you begin

* At least one data container (mounted volume) must be available to create a workspace.

.Steps

. Create a workspace:
.. In System Manager, navigate to *AI Data Engine > Workspaces*.
.. Select the *Add* button to open the workspace creation dialog.
.. Select at least one available data container (mounted volume) to associate with the workspace.
.. (Optional) Choose guard rails from the dropdown. If a guard rail is disabled or in test mode, review any warning messages.
.. Select *Continue*.
.. Enter a workspace name and description.
.. Select *Add* to create the workspace.

. View the workspace:
.. After creation, the workspace appears in the Workspaces inventory page.
.. Review workspace details, including state, total size, and percentage of cluster capacity used.
.. Select the workspace name to open the details page.
.. In the Overview tab, view workspace details and associated data containers.

. Add a data collection to the workspace:
.. In the workspace details page, go to the Data Collection tab.
.. Select the *Add* button to open the data collection dialog.
.. Enter the required information for the new data collection.
.. Select *Add* to associate the data collection with the workspace.
