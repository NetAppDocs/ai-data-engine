= Install the AFX storage system for AI Data Engine
:icons: font
:imagesdir: ../media/

[.lead]
Install the AFX storage system as the first step in deploying the AI Data Engine. The AFX storage system provides the storage infrastructure foundation and is required before installing the data compute nodes.

Follow the link:https://docs.netapp.com/us-en/ontap-afx/install-setup/install-setup-workflow.html[AFX 1K installation documentation^] to install the AFX storage system. 


.What's next

After completing the AFX storage system installation, link:install-workflow.html[install the data compute nodes].
